Hexo Next搭建个人博客系列一，初始化搭建


题注： 博客是在空闲的时间一点点搭建的，后来在想搭建过程的时候，居然有一些过程快要忘记了，虽然网上已经有了很多的搭建教程，还是再此整理一下，记录下整个Blog的搭建过程，希望能帮助更多的人搭建自己的Blog。

安装Git

下载并安装Git

Git命令在里面用到的不是很多，如果需要的话，您可以在安装Git之前需要了解下Git的一些基本命令。

安装Node.js

下载并安装Node.js

Node.js 主要用于Hexo的安装

这里在安装Hexo的时候遇到一个小问题，我的电脑是window7 64位操作系统，下载了64位的Node.js，在安装Hexo的时候总是报错，在网上搜索了各种资料后也没有解决，后来用32位的Node.js试了下就可以了。安装时的错误没留下，如果您遇到了问题安装不成功，可以试一下32位的Node.js。

安装Hexo

这是搭建博客最重要的一个步骤，先介绍几个命令

  1. hexo clean 清除生成的缓存静态文件
  2. hexo g 生成静态文件
  3. hexo s 启动本地预览
  4. hexo d 文件发布到Github或其他地方
  5. hexo n 新建文件

鼠标右键 Git Base here，在打开的命令窗口中使用一下命令安装Hexo

$ npm install hexo-cli -g

如果在安装过程中，出现以下错误

ERROR Deployer not found : github

则运行以下命令，或在安装前就运行此命令

$ npm install hexo-deployer-git --save

F盘新建Blog文件夹，文件路径可以任意，但最好不要有中文，进入文件夹后，鼠标右键 Git Base here，执行下面命令

$ hexo init

Hexo会自动将博客搭建的所有文件下载到Blog文件夹下

安装依赖包

$ npm install

至此，Hexo Next 博客的基本结构已经出现了，
在命令窗口中执行

$ hexo g
$ hexo s

浏览器访问http://localhost:4000/,此时博客基础架构就已经搭建完成了。

部署到GitHub

修改## Blog ##目录下的 _config.yml 配置文件，在最下方添加

deploy:
  type: git
  repo:
      github: git@github.com:shaipxiang/shaipxiang.github.io.git,master
      coding: git@git.coding.net:zhaojinghua/zhaojinghua.git,master

上面我添加了GitHub和coding两个托管地址，你在配置的时候可以先将coding的地址去掉，以后再添加，github在国内访问速度比较慢，coding国内访问比较快，所以托管了两个，以后如果购买域名后，可以两个都配置上。
注意修改成自己的github和coding地址，另外格式按照上面我的缩进格式，：后面多留一个空格，否则编译的时候会报错。

执行：

$ hexo g
$ hexo d

执行过程可能需要你输入github的邮箱和密码，执行成功后就讲文件都托管到了Github上了‘
访问 shaipxiang.github.io就能看到自己的博客了。

新建文章

在Blog文件夹中右键 Git Base Here，在命令窗口中执行

$ hexo new "my new post"

在Blog\source\_post文件件中出现my-new-post.md文件，可以使用Editplus或者MarkdownPad 2等文件进行修改。

修改完成后执行：

$ hexo g
$ hexo s   #本地预览
$ hexo d   #托管到github

至此，Hexo Next个人Blog就已经搭建出来了，如果大家有不清楚的，可以给我留言我们共同探讨。
